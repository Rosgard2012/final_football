Приклад методу на C#, який додає два числа та повертає їх суму:

public int Sum(int a, int b)
{
    return a + b;
}
Цей метод приймає два цілі числа (a та b) та повертає їх суму. 
Використовуйте його таким чином:

int x = 5;
int y = 7;
int result = Sum(x, y);
Console.WriteLine(result); // виведе 12

Якщо потрібно, щоб метод повертав результат з плаваючою крапкою, то 
замість int  можно використовувати double, float або будь-який інший тип 
даних з плаваючою крапкою. Наприклад, ось як виглядатиме метод, який 
повертає суму двох чисел з плаваючою крапкою:

public double Sum(double a, double b)
{
    return a + b;
}
